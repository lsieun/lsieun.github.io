## Bookmark

查看：

- `Shift + F11`: Show bookmarks
- `Alt + 2`: Bookmarks tool window

切换：

- `F11`: Toggle bookmark
- `Ctrl + F11`: Toggle bookmark with mnemonic
- `Ctrl + Shift + #[0-9]`: Toggle numbered bookmark

跳转：

- `Ctrl + #[0-9]`: Go to numbered bookmark
