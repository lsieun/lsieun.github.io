## Java 代码样式

```text
Settings --> Editor --> Code Style --> Java
```

- Wrapping and Braces
    - Brace placement
        - In class declaration: Next line
        - In method declaration: Next line
    - 'if()' statement
        - 'else' on new line
    - 'try' statement
        - 'catch' on new line
        - 'finally' on new line
