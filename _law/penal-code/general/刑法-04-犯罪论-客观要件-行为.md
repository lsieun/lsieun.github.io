## 不作为

### 不作为犯分类

```plantuml
@startmindmap

* 行为
    * 作为
        * “持有”属于作为
    * 不作为
        *[#lightgreen] 真正不作为犯
            * 丢失枪支不报罪
            * 遗弃罪
            * 拒不支付劳动报酬罪
            * 拒不执行判决、裁定罪
        *[#lightgreen] 不真正不作为犯

@endmindmap
```

### 不作为犯成立条件

```plantuml
@startmindmap

* 不作为犯成立条件
    * 客观条件
        *[#lightgreen] 应为（义务）
            * 危险源 → 监管义务
            * 法益对象 → 保护义务
        *[#lightgreen] 能为（能力）
            * 自身能力
            * 客观条件
        *[#lightgreen] 不为（无视）
            * 排除
                *[#orange] 结果避免可能性
            * 等价性问题（程度问题）
                *_ 母亲不给孩子喂养1小时
                *_ 母亲不给孩子喂养3天
            * 存在既遂和未遂
                *_ 母亲不给孩子喂养2天，孩子奄奄一息，邻居发现，马上救援，是未遂
    * 主观要件
        * 故意
            *[#lightgreen] 事实认识错误
            *[#lightgreen] 法律认识错误
        * 过失

@endmindmap
```
